# Wechaty ChatGPT

This is a wechaty implementation of ChatBot powered by OpenAI ChatGPT

## Setup

Modify `dotenv` file to add your openai_api_key as well as the service_token
```
OPENAI_API_KEY=""
WECHATY_PUPPET_SERVICE_TOKEN=""
```

```shell
npm install
mv dotenv .env
```

## Run

```shell
npm run start
```
